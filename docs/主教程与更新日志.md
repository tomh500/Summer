 ## 💾 安装方式  
 1. 将项目放入 CS2 的 CFG 文件夹中，文件夹名为 `Autumn`  
 2. 确保复制的是整个项目文件夹而非内部文件  
 3. 运行 `CFG主程序.bat`  
 4. 点击左上角 → 文件 → 更新并安装 CFG  
 5. 配置完成后点击打开用户空间并保存  


 启动项示例：  
 ```bash
 +exec Autumn/setup
 ```
 或由安装器自动添加到 `autoexec`（注意只能加载一次）

 ---

 ## 🗑️ 如何卸载 CFG  
 在主程序左上角文件菜单中删除
 或者
 1. 删除 `autoexec.cfg` 中的 `exec Autumn/setup`  
 2. 删除游戏启动项中的 `+exec Autumn/setup`  
 3. 控制台输入 `binddefaults` 重置绑定

 ## ❓ Q&A

 **Q：是外挂吗？**  
 A：不是，仅使用 CS2 官方指令，纯 CFG 项目  

 **Q：为什么项目里有 .exe？**  
 A：用于安装/卸载/验证，不操作游戏进程，安全无毒  

 **Q：为什么有 dll？**  
 A：用于安装模块调用，与 CS 游戏无关
